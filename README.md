# pridecons

A client-side web tool for generating customized pride-themed profile pictures with privacy and precision.

<img width="1919" height="1079" alt="image" src="https://github.com/user-attachments/assets/417113c0-1940-4c62-9c9b-d14ded6b9715" />


## tech stack

*   **Frontend:** HTML5, CSS3, JavaScript (Canvas API)
*   **Backend:** Python (Flask) - *used only for serving static files*

## local setup

1.  Clone the repository.
2.  Install the required dependency (Flask):
    ```bash
    pip install flask
    ```
3.  Run the application:
    ```bash
    python app.py
    ```
4.  Open your browser to `http://127.0.0.1:5000`.
